博客技术栈

博客框架与主题

Hexo框架

Hexo 是一个快速、简单且强大的博客框架，使用 Node.js 编写。它使用 Markdown 解析文章，几秒内即可生成静态网页。

Butterfly主题

本站使用 Butterfly 主题，一个优雅美观、功能丰富的主题。

主要插件

  • hexo-abbrlink：生成文章的短链接
  • hexo-blog-encrypt：文章加密功能
  • hexo-butterfly-tag-plugins-plus：增强型标签插件
  • hexo-deployer-git：Git 部署工具

特色功能

网络工具集

🌐 网络诊断与分析工具

本博客集成了一系列实用的网络诊断和分析工具，帮助您检测网络连接、查询IP信息并进行基础的网络测试。

IP 信息查询

快速查询任意IP地址或域名的详细信息，包括地理位置、ASN、运营商等关键数据。支持风险评估和代理检测功能。

Ping 测试工具

直观地测试网络连接质量，支持自定义测试次数和间隔，实时显示延迟和丢包率统计。

JSON 工具箱

提供JSON数据处理的全套功能：

  • 深度/浅层合并JSON数据
  • 格式化与压缩
  • 语法验证
  • 结构比较
  • 数据可视化
