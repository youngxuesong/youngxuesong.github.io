📺 项目简介

LibreTV 是一个轻量级、免费的在线视频搜索与观看平台，提供来自多个视频源的内容搜索与播放服务。无需注册，即开即用，支持多种设备访问。项目结合了前端技术和后端代理功能，可部署在支持服务端功能的各类网站托管服务上。

特点：

  • 完全免费使用
  • 多平台视频资源聚合
  • 操作简单，界面友好
  • 支持手机、平板、电脑等多种设备
  • 无需安装，随时访问

📋 详细部署指南

Cloudflare Pages 部署步骤

  1. 准备工作

    • 注册 GitHub 账号
    • 注册 Cloudflare 账号

  2. 获取项目源码

    • Fork 或克隆 LibreTV 仓库到您的 GitHub 账户
    • 仓库链接：LibreTV 官方仓库

  3. 配置 Cloudflare Pages

    • 登录 Cloudflare Dashboard
    • 进入 Pages 服务
    • 点击”创建项目”，选择”连接到 Git”
    • 授权并选择您 Fork 的 LibreTV 仓库

  4. 部署设置

    • 项目名称：自定义（例如 “my-libretv”）
    • 构建设置：
      • 构建命令：留空（无需构建）
      • 输出目录：留空（默认为根目录）
    • 环境变量设置（非常重要）：
      • 在”设置” > “环境变量”中添加 PASSWORD 变量（设置您的访问密码）
      • 可选：添加 ADMINPASSWORD 变量（管理员密码，用于更高级功能）

  5. 完成部署

    • 点击”保存并部署”
    • 等待部署完成，Cloudflare 会提供一个 *.pages.dev 的访问链接
    • 您也可以绑定自己的域名（在 Cloudflare Pages 的”自定义域”设置中）

💡 使用技巧

  1. 搜索影片

    • 在搜索框输入关键词，点击搜索按钮
    • 支持按类型、年份等筛选内容

  2. 播放控制

    • 支持暂停、快进、音量调节等基本功能
    • 部分内容支持选择清晰度

  3. 收藏功能

    • 可以将喜欢的影片添加到收藏夹
    • 支持创建自定义播放列表

  4. 多设备同步

    • 账号登录后可在不同设备间同步观看历史和收藏

⚠️ 注意事项

  • LibreTV 仅提供搜索和播放功能，不存储任何视频内容
  • 所有视频资源来自第三方网站，请注意甄别内容安全性
  • 建议设置较强的访问密码，避免他人未授权使用
  • 定期更新 API 接口，保证视频资源的可用性

🔍 常见问题解答

Q: 为什么有些视频无法播放？
A: 可能是源站资源失效或接口限制，尝试更换其他视频源或更新 API 接口。

Q: 如何添加自定义视频源？
A: 在管理员设置中，可以添加符合格式要求的第三方 API 接口。

Q: 部署后如何更新 LibreTV？
A: GitHub 仓库更新后，可以在 Cloudflare Pages 中触发重新部署来更新您的站点。
